= 在 VCF 管理域中添加 iSCSI 数据存储作为补充存储的部署工作流程
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
开始添加 iSCSI 数据存储作为 VMware Cloud Foundation (VCF) 管理域的补充存储。您将设置一个具有用于 iSCSI 的逻辑接口 (LIF) 的存储虚拟机 (SVM)，在 ESXi 主机上配置 iSCSI 网络，ONTAP tools for VMware vSphere，并创建 VMFS 数据存储库。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一个"]link:vmw-vcf-mgmt-supplemental-iscsi-requirements.html["查看部署要求"]
[role="quick-margin-para"]
查看将 iSCSI 数据存储作为补充存储添加到 VCF 管理域的要求。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["两个"]link:vmw-vcf-mgmt-supplemental-iscsi-svm-lifs.html["创建 SVM 和 LIF"]
[role="quick-margin-para"]
为 iSCSI 流量创建具有多个 LIF 的 SVM。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三"]link:vmw-vcf-mgmt-supplemental-iscsi-network.html["配置网络"]
[role="quick-margin-para"]
在 ESXi 主机上设置 iSCSI 网络。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四个"]link:vmw-vcf-mgmt-supplemental-iscsi-storage.html["配置存储"]
[role="quick-margin-para"]
部署并使用ONTAP工具配置存储。
